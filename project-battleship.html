AI Battleship Game

This project involved creating an adaptive AI to play the game Battleship. I implemented logic using HeatMaps and NumPy to optimize search patterns and reduce the average number of moves per game.

  • Tech Used: Python, NumPy
  • Challenges: Efficient hit-tracking and reducing guess redundancy
  • Outcome: Reduced average moves per game from 80 to ~67
